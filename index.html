What it does

Calculates real electrical load per server and shows how it distributes across PDU banks and phases.

  • Total amperage per server
  • Amperage per feed
  • Bank load

How it calculates

Uses actual server characteristics instead of theoretical peak values.

  • PSU count
  • PSU wattage
  • Voltage
  • Utilization level

Why it matters

Breakers trip because of imbalanced load, not because of one big server.

  • Prevent overcurrent
  • Balance phases
  • Plan rack capacity
  • Design resilient layouts
